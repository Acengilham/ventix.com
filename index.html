Dashboard Sistem Event

Kelola event Anda dengan mudah menggunakan sistem kami. Generate QR Code untuk peserta, pantau kehadiran, dan daftarkan peserta on the spot.

Generate QR Code

Buat QR Code unik untuk event Anda yang bisa digunakan peserta untuk check-in. Mudah, cepat, dan efisien.

Event Check-in

Pantau kehadiran peserta event secara real-time. Scan QR Code peserta dan dapatkan data kehadiran lengkap.

Pendaftaran OTS

Daftarkan peserta yang datang langsung (on the spot). Input data peserta dan langsung generate QR Code-nya.

Panduan Penggunaan

Generate QR Code

1

Masuk ke Menu Generator

Klik tombol "Buka Generator" pada card Generate QR Code untuk masuk ke halaman generator.

2

Masukan Daftar Nama Peserta

Anda dapat memasukan daftar nama peserta untuk generate QR Code massal.

3

Generate QR Code

Klik tombol "Generate" untuk membuat QR Code. Tunggu beberapa saat hingga proses selesai.

4

Download QR CODE

Setelah QR Code siap, Anda bisa mendownload semua hasil bulk dalam format PNG/JPG .

Event Check-in

1

Akses Menu Check-in

Klik tombol "Buka Check-in" pada card Event Check-in untuk masuk ke halaman check-in.

2

Pilih Event

Pilih event yang ingin Anda pantau dari daftar event yang tersedia.

3

Scan QR Code Peserta

Gunakan scanner kamera untuk scan QR Code yang dibawa peserta atau input manual kode unik peserta.

4

Verifikasi Data

Sistem akan menampilkan data peserta. Verifikasi kebenaran data sebelum melanjutkan.

5

Selesaikan Check-in

Klik tombol "Check-in" untuk mencatat kehadiran peserta. Data akan tersimpan otomatis.

6

Pantau Laporan

Anda bisa melihat laporan kehadiran secara real-time di dashboard atau export data ke Excel.

Pendaftaran On The Spot

1

Akses Menu Pendaftaran OTS

Klik tombol "Buka Pendaftaran" pada card Pendaftaran OTS untuk masuk ke halaman pendaftaran.

2

Pilih Event

Pilih event yang ingin Anda buka pendaftaran on the spot-nya.

3

Isi Formulir Pendaftaran

Input data lengkap peserta yang datang langsung, termasuk nama, email, nomor telepon, dan informasi lainnya yang diperlukan.

4

Generate QR Code Instan

Setelah data disimpan, sistem akan langsung generate QR Code untuk peserta tersebut.

5

Cetak atau Kirim QR Code

Anda bisa langsung mencetak QR Code atau mengirimkannya via email kepada peserta.

6

Peserta Dapat Langsung Check-in

Peserta yang sudah terdaftar bisa langsung melakukan check-in menggunakan QR Code yang baru dibuat.
